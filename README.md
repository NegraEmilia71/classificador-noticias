# Classificador de Categorias de Notícias

## 🎯 Objetivo
API de classificação automática de notícias brasileiras por categoria.

## 📊 Dataset
News of the site FolhaUOL (Kaggle)
https://www.kaggle.com/datasets/marlesson/news-of-the-site-folhauol

## 🛠️ Tecnologias
- Python 3.9+
- Scikit-learn (modelagem)
- FastAPI (API)
- Docker (containerização)
